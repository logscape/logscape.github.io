Logscape in the Cloud

Logscape now provides enterprise level monitoring, search and root cause analysis in the cloud. Users can take advantage of team management features, build custom workspaces and searches.

The Logscape Cloud services accepts data from many different sources types. Linux clients and network devices can export their data via the syslog transport. Windows clients have the option of using nxlog to ship logs into the cloud. Open Source log shippers can also be used in conjunction with Logscape.

Once you have signed up check out the following guides to get data in to Logscape.
