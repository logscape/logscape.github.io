Logscape Monitoring Agent Health

Agent Count

THe chart displays the total number of agents in your deployment as a line chart overlayed over the number of Agents joining your deploment. When Agents are shutdown and restarted you will see this chart change. Use this chart to see if you have all the expected agents in the Logscsape deployment.

- Agent Services

A Service is a task executed by Agent either as part of an App or as a system service. This displays the status of services running in your system. Use this chart to debug Apps running in your environment.

- System Load

This is the process load of each Agent running in your system. An Agents processor load is affected by its role. An Agent can either be

  • Manager
  • IndexStore
  • Forwarder
  • Indexer
Managers,IndexStores and Indexer all have ingestion and indexing functionality. Expect CPU load to be related to the amount of data being searched and indexed. The Manager's load is also affected by the size of the deployment and the activity of the system.
