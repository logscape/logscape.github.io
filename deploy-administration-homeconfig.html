    Customizing the Home Page

    To customize the Home Page ,click on the Menu Bar and click on Home

    Then once the Home Page has loaded. Click on the edit button

    Customize the Home Page by adding charts, searches and links to documentation and ticket system relevant to your users.

    Navigate to the Deploy page and type Home in the filter and then click export

    Save the file as Logscape-Home.config

    When ever you deploy Logscape you can use this file to customize your Logscape Home Workspace.
