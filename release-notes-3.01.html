Release, 3.01

Nov 3 2015

Logscape 3.01 Is a small patch in order to assess a number of issues highlighted by the 3.0 release, it is reccomended all users upgrade 3.01 at their earliest convenience.

Team Page

Restored functionality to the Team page after issues were introduced during the translation process.

Data Types Page

Fixed an issue whereby datatype fields could become corrupt, resulting in the datatypes page being unable to render them.

Overflowing Pie Charts

CSS Styling has been updated to prevent D3 pie charts overflowing their widget in Internet Explorer.

Event Popup

Fixed an issue whereby facets could occasionally return an incorrect result resulting in the Event Popup rendering the incorrect event. Also slight styling changes.

Overscan Error

Fixed an issue with Multi-line events which caused Overscan errors.

Datatables Behaviour

Fixed an issue where Datatables would become locked with a processing message.

Uptime

Re-added the Uptime message on the system page.

LDAP

Fixed an issue where LDAP settings could not be saved.
