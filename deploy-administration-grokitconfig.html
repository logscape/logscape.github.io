Grokit Properties File

The Grokit file extracts predifned expressions into fields. These fields are available at search time. Use the grokit properties file to determine commonly occuring patterns across all log types. Examples e.g url,paths, exceptions and so on. Each entry in the file has the following pattern.

$FIELDNAME:$PATTERN

The $FIELDNAME will appear in the discovered fields section of the search page and can be used in searches in the same way fields and discovered fields are.

See the sample grokit file below: 

#File masks - regular expression
fileIncludePath:.*.log.*
fileExcludePath:
#field-name and regular expression matchers that extract a single group for the value
_email:.*?([_A-Za-z0-9-\.]+@[A-Za-z0-9-]+\.[A-Za-z]{2,}).*?
_ipAddress:.*?([0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}).*?
_exception:.*?([_A-Za-z0-9-\.]+Exception).*?
_url:.*?([A-Za-z]{4,4}://[A-Za-z.0-9]+[:0-9]{0,6}[A-Za-z/]+).*?
_level:.*?(INFO|ERROR|WARN|DEBUG|FATAL|TRACE|SEVERE|DEBUG).*?
_hour:.*?[,.\s-]([0-9]{2,2}):[0-9]{2,2}:[0-9]{2,2}[,.\s-].*?
#_minute:.*?[,.\s-][0-9]{2,2}:([0-9]{2,2}):[0-9]{2,2}[,.\s-].*?
