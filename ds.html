    Overview

    Logscape uses Data Sources to import data into the system. As data is ingested common patterns from the data are extracted and become available at search time. There are several ways of getting data into Logscape, from pointing to files on the filesystem, apps or ingesting data from syslog clients and raw tcp clients.
