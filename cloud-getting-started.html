How does it work?

When you send your logs to the Logscape cloud they get preprended with a tag and a security token, this token is unique to you, not only does it allow your data into the Logscape system, it also guarentees that only you will be able to see it, once in the cloud your data is written to disk and then ingested by Logscape so that it can be searched by you.
