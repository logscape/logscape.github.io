Manager Installation

The Manager monitors and visualizes your data and also runs a Syslog Server (udp:1468;tcp:1516).

On Unix

Download Logscape-{VERSION}.zip from the download page and then run "logscape.sh start".

On Windows

Download the MSI and choose the Manager Role. The package will automatically install and start the Logscape Agent service.

Manager Install

Forwarder Installation

You can now install Forwarders on several hosts to collect data from your applications and devices. The Forwarders stream your data to the Manager for indexing and search.

On Unix

Download the Logscape-{VERSION}.zip on a host. Unzip the archive and anvigate to the following folder

$LOGSCAPE_HOME/scripts

To configure the Logscape Agent as a Forwarder, run the configure.sh and then choose the "Forwarder" role. Once the setup has completed start the Forwarder by typing:

./logscape.sh start

On Windows - Logscape.MSI:

Copy the Logscape MSI to a new host and choose the "Forwarder" Role. The installer will automatically run the Forwarder as a service.

Forwarder Install
